# 妆搭 Makeup Mate PRD（三页架构版）

> 项目名称：妆搭 Makeup Mate
> 产品定位：会记住你的 AI 美妆复刻陪练
> 核心一句话：把刷到、看到或选中的妆容，变成适合你的专属上妆教程。
> 页面架构：底部导航仅保留 **首页 / 聊天 / 我的** 三个页面。
> 版本：V1.0

---

## 一、简介

### 1.1 项目背景

抖音、小红书等平台上已经沉淀了大量美妆教程内容，但用户真实复刻时存在明显断点：视频看起来很简单，实际操作时却常常出现“看不懂步骤、找不到重点、不知道适不适合自己、化到一半没人指导”等问题。

用户的典型行为是：刷到美妆视频后收藏、截图、反复暂停，但最终仍然难以把视频内容转化为自己的上妆流程。尤其对新手用户而言，美妆视频只是内容消费，并没有真正变成可执行的生活方案。

**妆搭 Makeup Mate** 的目标是解决这个断点：

> 将美妆视频、图片和达人风格重构为结构化、个性化、可陪伴执行的上妆教程。

---

### 1.2 产品定位

**妆搭 Makeup Mate** 是一款 AI 美妆复刻陪练小程序。用户在首页粘贴美妆视频链接，或上传妆容图片、本地短视频片段后，系统自动解析内容来源中的妆容风格、关键步骤、产品类型、难度和耗时，并生成一张可分享、可导入聊天的“妆容解析卡片”。

用户进入聊天页后，可以基于该妆容解析卡片与 AI 开启文字、语音或视频陪伴上妆。聊天页同时是产品的 **美妆灵感库**：用户没有明确链接时，也可以从美妆博主库、风格复刻、场景妆容和新手陪练开始，选择一个达人或风格生成自己的复刻方案，再进入对话。聊天页右上角提供两个核心入口：**历史记录**和**上传照片**。用户可以上传自拍，AI 分析个人风格、脸型、肤色、五官特点，并把这些结果沉淀到“我的妆容档案”中。后续 AI 会根据用户的妆容档案、历史偏好和操作反馈，持续优化妆容复刻建议。

---

### 1.3 核心理念

本项目不是简单的“AI 识别妆容”，也不是“虚拟试妆工具”，而是：

> **美妆内容重构 + 个人风格适配 + 陪伴式上妆执行 + 长期妆容记忆。**

产品重点强调四件事：

1. **内容可执行化**：把视频、图片和达人风格拆成步骤、产品、时间和注意事项。
2. **建议个性化**：根据用户照片和妆容档案改写原教程。
3. **过程陪伴化**：通过聊天、语音、视频模式陪用户一步步上妆。
4. **档案长期化**：记住用户适合什么、不适合什么、哪里容易翻车。

## 二、竞品分析

### 2.1 竞品类型概览

当前市场中，与妆搭相关的产品主要可以分为四类：

| 类型           | 代表产品                                        | 核心能力                     | 局限                               |
| -------------- | ----------------------------------------------- | ---------------------------- | ---------------------------------- |
| 修图美颜类     | 美图秀秀、醒图                                  | 图片精修、人像美颜、妆容模板 | 偏“修图结果”，不解决真实上妆过程 |
| AI 测肤/测脸类 | 美图宜肤、美的你                                | 肤质分析、脸型测试、色彩测试 | 偏测评和推荐，缺少视频教程复刻闭环 |
| AR 试妆类      | YouCam Makeup、ModiFace、Sephora Virtual Artist | 虚拟试色、AR 妆效预览        | 偏电商导购，较少关注用户真实操作   |
| 内容平台类     | 抖音、小红书、B站                               | 大量美妆教程内容             | 视频内容非结构化，用户复刻成本高   |

---

### 2.2 主要竞品对比

| 产品          | 可借鉴点                         | 与妆搭的差异                                     |
| ------------- | -------------------------------- | ------------------------------------------------ |
| 美图秀秀      | 人像美化、视频美颜、AI 生成能力  | 美图偏“修出好看结果”，妆搭偏“教用户真实复刻” |
| 醒图          | 简洁视觉、局部调整、自然美型     | 醒图偏图片编辑，妆搭偏视频内容重构和过程陪伴     |
| 美的你        | 脸型分析、肤色分析、个人风格推荐 | 美的你偏测评，妆搭把测评结果用于视频教程改写     |
| YouCam Makeup | 虚拟试妆、妆效预览、AI 美妆工具  | YouCam 偏试妆编辑器，妆搭偏 AI 美妆陪练          |
| ModiFace      | AR 试色、品牌商品连接            | ModiFace 偏品牌导购，妆搭偏内容复刻和个性化执行  |
| 抖音美妆视频  | 海量教程、趋势内容、用户兴趣强   | 抖音负责内容消费，妆搭负责内容转化和陪伴执行     |

---

### 2.3 竞品机会点

现有产品大多解决以下问题：

1. 如何把照片修得更好看；
2. 如何在线试一个妆效或色号；
3. 如何测试脸型、肤色、个人色彩；
4. 如何浏览更多美妆内容。

但它们没有完整解决一个高频场景：

> 用户刷到一个美妆视频后，如何知道这个妆适不适合自己？如何拆出具体步骤？如何把原教程改写成适合自己的版本？如何在真实化妆过程中获得陪伴和纠错？

这就是妆搭的差异化机会。

---

### 2.4 妆搭的差异化定位

| 维度     | 传统产品                     | 妆搭 Makeup Mate                                |
| -------- | ---------------------------- | ----------------------------------------------- |
| 核心对象 | 图片、人脸、商品色号         | 美妆视频、图片、达人风格与场景妆容              |
| 用户动作 | 修图、试色、测肤             | 粘贴链接、上传图片/视频、导入聊天、跟着 AI 上妆 |
| AI 角色  | 修图工具、识别工具、导购工具 | 内容拆解员、个人化妆师、陪练搭子                |
| 输出结果 | 精修图、试妆图、商品推荐     | 妆容解析卡片、个性化步骤、陪伴上妆记录          |
| 长期价值 | 模板复用                     | 记住用户妆容档案和偏好                          |

妆搭不与美图、醒图正面竞争“谁修图更自然”，也不与 AR 试妆工具竞争“谁试色更逼真”。妆搭建立的是新的产品心智：

> **AI 美妆视频复刻陪练。**

## 3.2 首页：解析入口与妆容卡片生成

### 3.2.1 页面目标

首页是产品的启动页，只保留一个任务：

> 输入一个美妆内容来源，生成一张可执行的“妆容解析卡片”。

首页不放“抖音美妆灵感库”或内容广场，避免分散主链路。用户一进入首页，就应该知道可以粘贴链接、上传图片或上传视频，并得到自己的专属复刻卡片。

首页核心文案：

> 粘贴美妆视频链接，或上传图片/视频，生成你的专属复刻卡片。

首页只做四件事：

1. 粘贴美妆内容链接，V1 重点支持抖音，预留小红书、B站等平台；
2. 上传妆容图片，例如截图、封面图、博主妆容图；
3. 上传本地短视频片段，系统抽帧分析；
4. 生成妆容解析卡片，并支持分享或导入聊天页。

## 3.3 聊天页：AI 陪伴与美妆灵感库

### 3.3.1 页面目标

聊天页不只是聊天窗口，而是产品的 AI 陪伴与美妆灵感中心。它承担四个功能：

1. 接收首页导入的妆容卡片；
2. 通过文字、语音、视频陪伴用户上妆；
3. 承载美妆博主库、风格复刻、场景妆容和新手陪练；
4. 通过右上角入口查看历史记录、上传照片分析个人风格。

用户没有明确视频链接时，也可以从聊天页的美妆灵感库开始探索。这里不是普通内容广场，而是“可对话、可复刻、可生成我的版本”的美妆风格入口。

包装方式：

> 每个美妆博主或妆容风格，都是一个可对话的妆容助手。

---

聊天页参考用户提供的界面风格，采用深色背景、荧光绿色点缀、卡片化入口和底部输入框。页面上半部分是美妆灵感库，下半部分仍保留聊天能力。

### 3.3.3 美妆灵感库

美妆灵感库放在聊天页有三个原因：

1. 首页更清爽，只保留解析入口，不分散用户注意力；
2. 聊天页更有内容感，用户没有明确链接时，也可以从达人/风格开始探索；
3. 更符合 AI 产品逻辑，用户不是浏览普通内容，而是选择一个风格助手或达人风格助手，然后进入对话和复刻。

美妆灵感库包含四类入口：

| 入口       | 说明                                             |
| ---------- | ------------------------------------------------ |
| 美妆博主库 | 按达人风格组织，可选择达人并生成自己的复刻方案   |
| 风格复刻   | 按妆容风格组织，例如清冷通勤、甜妹约会、港风复古 |
| 场景妆容   | 按使用场景组织，例如上课、通勤、面试、拍照       |
| 新手陪练   | 面向新手的低难度、短步骤、低翻车风险妆容方案     |

示例入口：

- 清冷通勤妆助手；
- 甜妹约会妆助手；
- 港风复古妆助手；
- 泰妆上镜妆助手；
- 新手 10 分钟妆助手；
- 某某达人风格助手。

### 3.3.9 个性化改写

当用户已有个人妆容档案或上传照片后，AI 将原视频教程改写为“我的版本”。

| 原视频步骤         | 我的版本                           |
| ------------------ | ---------------------------------- |
| 上挑眼线拉长 5mm   | 改为自然延长 2mm，更适合淡颜和通勤 |
| 腮红扫在颧骨下方   | 改为眼下外侧上移，更适合方圆脸     |
| 深豆沙唇泥         | 改为奶茶豆沙色，降低成熟感         |
| 鼻影从眉头连到鼻尖 | 改为只轻扫山根和鼻头，避免显脏     |

### 3.3.11 陪伴上妆步骤

| 步骤        | AI 指导                                        |
| ----------- | ---------------------------------------------- |
| Step 1 底妆 | 先完成轻薄底妆，重点是均匀肤色，不追求强遮瑕   |

| Step 6 唇妆 | 薄涂奶茶豆沙色，边缘用手指晕开                 |
| Step 7 检查 | 检查眼线是否过长、腮红是否过低、整体是否干净   |

## 八、产品边界与合规

### 8.1 不做的功能

V1 不做以下内容：

- 颜值评分；
- 医疗诊断；
- 皮肤病判断；
- 容貌缺陷评价；
- 强商品导购；
- 大而全社区；
- 美甲、穿搭、发型等泛化功能；
- 精准人脸识别或身份识别。

---

### 8.2 表达边界

错误表达：

> 你的脸太圆，眼睛不够大，颜值偏低。

正确表达：

> 你的脸型偏圆，更适合把腮红位置上移到眼下外侧，视觉上更轻盈；眼妆建议选择短眼线和浅色晕染，保留干净感。

---

### 8.3 隐私原则

产品应遵守以下原则：

1. 默认不保存原始自拍和视频；
2. 只保存用户授权后的结构化妆容档案；
3. 用户可选择“仅本次分析”；
4. 用户可随时删除档案和历史记录；
5. 不做人脸识别登录；
6. 不做人脸比对；
7. 不建立人脸库；
8. 不输出颜值评分和容貌羞辱内容。

## 九、最终版本一句话

**妆搭 Makeup Mate** 是一款三页结构的 AI 美妆复刻陪练小程序。首页负责把美妆视频链接、妆容图片或本地短视频解析成可分享、可导入聊天的妆容解析卡片；聊天页负责承载美妆灵感库、接收妆容卡片，并通过文字、语音、视频陪伴用户真实上妆，同时支持历史记录和自拍分析；我的页负责沉淀用户的妆容档案、AI 已记住、历史复刻和隐私设置。产品的核心价值不是“浏览更多美妆内容”，而是让用户真正把刷到、看到或选中的妆容画到自己脸上。
